Respuestas

1. Estos son los componentes de una red y sus usos:

  1. Host (también llamados "equipos terminales")

    • Clientes: Tienen software que solicitan y muestran información obtenida de servidores
    • Servidores: Computadoras que poseen software que pueden proveer información (ej.: email, páginas, etc) a otros dispositivos en la red.
    • PCs: Como los servidores, son computadoras capaces de poseer software de servidores, y proveer información tal como un servidor. Son menos potentes que los servidores.
    • Otros equipos terminales:
      1. Impresoras
      2. Teléfonos IP
  2. Dispositivos intermedios
    • Conectan a los dispositivos terminales a la red
    • Proveen conectividad
    • Utilizan las direcciones de los dispositivos terminales para saber a dónde mandar los mensajes.

      • Ejemplos de dispositivos intermedios:

      1. Switch Capa 2
      2. Switch multicapa
      3. Router
      4. Router Wireless

2b. Modelo de referencia OSI:

Los modelos de referencia nos proveen con las herramientas necesarias para entender las relaciones entre los protocolos, y del lenguaje para describir las funciones y capacidades de la red de datos. También, brinda una lista de las funciones que deben ocurrir en cada capa.
